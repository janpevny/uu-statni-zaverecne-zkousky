# Frontendové systémy

### [1. Klasifikace frontendových technologií](1.md)
Co je to frontend. Webové aplikace, nativní mobilní aplikace, VR/AR/MR, aplikace pro smartwatch a smartTV apod.
### [2. JavaScript](2.md)
Rozdíly oproti backendu. Transpilační nástroje pro psaní nejnovějšího JS pro moderní prohlížeče.
### [3. Frontendové knihovy a komponentový přístup](3.md)
Moderní knihovny pro vývoj frontendových aplikací . Jejich vlastnosti a využití. Rozdělení prezentace do logických bloků (komponent) a jejich konfigurace a znovupoužitelnost.
### [4. Komunikace se serverem](4.md)
Způsoby načtení dat z backendu systému, transformace JSON dat do uživatelsky čitelné podoby.
### [5. Responzivita, stylování a UX](5.md)
Základní principy a vlastnosti CSS, responsivní web, CSS Frameworky a CSS preprocesory. Principy responzivity a mobile first přístup. Základy návrhu uživatelského rozhraní pro snadnou použitelnost a ovladatelnost. Způsoby rozmístění ovládacích prvků v aplikaci.
### [6. Uživatelské vstupy](6.md)
Formuláře a práce s nimi. Validace, dynamické formuláře, ukládání dat na backend.
### [7. Mockování](7.md)
Prototypování frontentu aplikace bez dostupnosti dat z backendu. Základní principy a způsob řešení.
### [8. Vstupní body aplikace](8.md)
Rozdělení aplikace podle vstupních bodů. Separace a sdílení funkčností mezi vstupními body aplikace.
### [9. Dokumentace a testování frontendu](9.md)
Způsoby dokumentace frontendové aplikace. Jak popisovat uživatelské rozhraní frontendové aplikace. Testovatelnost frontendové aplikace. Způsoby testování komponent a aplikace jako celku. Typy testů, proces testování (příprava testu, provedení a vyhodnocení), tvorba unit testů a zásady dokumentace zdrojového kódu.
### [10. Identita uživatele](10.md)
 Komunikace se serverem pod identitou uživatele. Správa identity ve frontendové aplikaci.
